Add an AI-powered chatbot to your WordPress website that learns from your content and helps visitors find information.

== Description ==

Website Chatbot is an intelligent assistant that helps your visitors find information and get answers about your website content. The chatbot uses OpenAI's GPT technology to understand and respond to questions based on your website's content.

= Key Features =

* Floating chat widget that's easy to access
* AI-powered responses based on your website content
* Full chat history tracking
* Admin dashboard for monitoring conversations
* Mobile-friendly design
* Customizable appearance
* Session management for continuous conversations
* Fullscreen mode support
* Clear chat history option

= Technical Features =

* OpenAI GPT-3.5 Turbo integration
* Automatic website content indexing
* Token optimization for API usage
* Secure data handling
* Cross-browser compatibility
* Responsive design
* AJAX-powered real-time chat
* Error handling and offline support

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/website-chatbot` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Website Chatbot settings page in your admin dashboard
4. Enter your OpenAI API key
5. The chatbot will automatically appear on your website

== Configuration ==

1. Get an API key from OpenAI (https://platform.openai.com)
2. Go to Website Chatbot settings in your WordPress admin panel
3. Enter your OpenAI API key in the settings
4. Save changes

== Frequently Asked Questions ==

= Do I need an OpenAI account? =

Yes, you need an OpenAI account and API key to use this plugin. You can get one at https://platform.openai.com

= How does the chatbot learn my website content? =

The chatbot automatically indexes your published posts, pages, and custom post types. It uses this content to provide context-aware responses to visitor questions.

= Is chat history stored? =

Yes, chat history is stored in your WordPress database for reference and analysis. You can view all conversations in the admin dashboard.

= Can I customize the chatbot's appearance? =

Yes, the chatbot's appearance can be customized through CSS. The plugin includes well-structured CSS classes for styling.

= Is it mobile-friendly? =

Yes, the chatbot is fully responsive and works well on all device sizes.
